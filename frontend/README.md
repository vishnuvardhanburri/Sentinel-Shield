# Sentinel Shield Frontend

Buyer-facing CISO dashboard for Sentinel Shield by Xavira Tech Labs.

## Local Dashboard

```bash
pnpm dev
```

Open:

```text
http://localhost:3000
```

The dashboard connects to the Sentinel Shield FastAPI gateway at:

```text
http://localhost:8000
```

Configure a different gateway with:

```bash
NEXT_PUBLIC_API_URL=https://your-sentinel-gateway.example.com
```
