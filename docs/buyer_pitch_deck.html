Sentinel Shield

Sovereign AI gateway for enterprises that need local AI, PII masking, prompt-injection defense, and board-ready compliance evidence.

The Problem

Employees are adopting AI faster than security teams can govern it. Sensitive identifiers, trade secrets, and regulated records can leak into models without strong controls or audit proof.

The Product

Identity Masking

Pseudonymizes Aadhaar, PAN, IFSC, UPI, GST, PHI, financial and custom PII.

Local AI Routing

High-risk prompts are forced to local Ollama models.

Obsidian Ledger

Hash-chained JSONL evidence for tamper detection.

Enterprise Proof

  • Oracle risk scoring and auto-quarantine.
  • Prompt injection and semantic DLP controls.
  • mTLS, scoped API keys, rate limits, and cost controls.
  • Readiness certificates, deployment pack, handoff ZIP, and CI artifacts.

How Buyers Verify

Run pnpm verify:buyer. The command checks compile, dashboard build, deployment doctor, API smoke, certificate generation, architecture PDF, and handoff ZIP.

Deployment

Local-first by default. Production hardening supports Nginx/Envoy mTLS, Redis risk state, encrypted backups, and buyer-owned immutable ledger anchoring.
